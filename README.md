# Power-BI-Workbook
# 📊 Power BI Retail & Sales Insights Dashboard  
### Data Technician Bootcamp — Power BI Project

This repository contains my Power BI project completed during the **Data Technician Bootcamp**, focused on transforming raw retail and sales data into an interactive, insight‑driven analytics report.  
The project demonstrates end‑to‑end BI skills including **data cleaning**, **data modelling**, **DAX**, and **visual storytelling**.

---

## 🚀 Project Overview

The goal of this project was to build a clear, compelling Power BI report that helps stakeholders understand key retail performance trends.  
Using Power Query, DAX, and a range of visualisation techniques, I created a multi‑page report that highlights:

- Sales performance across regions  
- Customer and product trends  
- Time‑based patterns in revenue  
- Category‑level comparisons  
- Interactive filtering for deeper exploration  

---

## 🧹 Data Transformation & Cleaning (Power Query)

Using **Power Query**, I prepared the dataset for analysis by:

- Removing errors and null values  
- Splitting and merging columns  
- Standardising data types  
- Creating new calculated fields  
- Filtering and shaping tables for modelling  

This step ensured a clean, reliable semantic model for downstream analysis.

---

## 🧠 DAX Calculations

I created **DAX calculated columns and measures** to support richer insights, including:

- Year‑to‑Date (YTD) and Month‑to‑Date (MTD) metrics  
- Total Sales, Profit, and Quantity measures  
- Category and region‑level aggregations  
- Percentage change and comparison metrics  

These calculations enabled dynamic, context‑aware visuals throughout the report.

---

## 📈 Visualisations & Report Design

The report includes a variety of visuals to support different analytical perspectives:

- **Bar charts** for category and region comparisons  
- **Line charts** for time‑series sales trends  
- **Pie charts** for proportional breakdowns  
- **Maps** to visualise geographic performance  
- **Cards & KPIs** for high‑level metrics  

Interactive elements include:

- **Slicers** for date, region, and product filtering  
- **Cross‑highlighting** between visuals  
- **Drill‑through pages** for deeper detail

## 🎨 Data Storytelling

The final dashboard focuses on clarity and narrative flow:

- Key KPIs are surfaced at the top for quick decision‑making  
- Visuals are grouped logically to guide the viewer  
- Colour palettes and layout follow best‑practice BI design  
- Insights emphasise *why* trends matter, not just *what* they are  

This approach transforms raw data into a meaningful story for retail stakeholders.
